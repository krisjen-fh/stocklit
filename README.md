<h1 align="center">📈 STOCKLIT</h1>

STOCKLIT is a simple and interactive stock analysis web app built with Streamlit and Plotly.
It helps users explore stock price movements, compare companies, and get basic buy/sell insights — all in a fun and visual way.
#
### ✨ Features
+ 🔍 Search stocks by company name
+ 📊 Candlestick chart for price trends
+ 📋 Stock data table
+ 🥧 Buyer vs Seller volume comparison
+ 📈 Automatic Buy / Sell / Hold indicators
+ 🔄 Compare two different companies
+ 🎨 Clean and interactive UI
#
### 🛠️ Tech Stack
+ Python
+ Streamlit
+ Plotly
+ Pandas
+ CSV dataset
#
### ▶️ How to Run
pip install streamlit plotly pandas

streamlit run app.py
#
### 📊 Data
Stock data includes: Date, Company Name, Stock Code, Open Price, Close, High, Low, Change, Offer Volume, Bid Volume

📅 Period: 2 October – 1 December 2023
#
### 🧠 Buy / Sell Logic
Change	Signal
> 0	Jual
< 0	Beli
= 0	Tahan
